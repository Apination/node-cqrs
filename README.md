NodeJS CQRS
===========

A set of backbone classes for CQRS app development

## Usage

```bash
npm install bitbucket:cqrs/node-cqrs --save
```

## Contribuion

Use editorconfig, eslint, `npm test -- --watch`


### Conventions

-	Constructors should take a single `options` argument
